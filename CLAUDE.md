# Teaching Repository

A teaching repository by Mark Pham for educating students on data-related topics.

## Repository Purpose

This repo contains teaching materials covering:

- **Statistics** — foundational concepts for data analysis
- **Data Pipelines** — ETL/ELT patterns, data flow architecture
- **Data Engineering** — infrastructure, orchestration, data modeling
- **SQL** — querying, window functions, CTEs, optimization
- **Python** — data manipulation, scripting, libraries (pandas, etc.)
- **BI Tools** — dashboarding, visualization, reporting
- **Best Practices** — data cleaning, code quality, documentation

## Repo Structure

```text
presentations/   — code-based slides (Marp, reveal.js, etc.)
notes/           — topic guides and written notes (markdown)
diagrams/        — architecture and flow diagrams (draw.io)
resources/       — books, references, external materials
```

## Content Guidelines

- Materials should be clear and accessible for students at varying levels
- Use practical examples and real-world scenarios
- Markdown files for written guides; draw.io for diagrams
- Keep explanations concise — lead with the "what" and "why", then the "how"

## Conventions

- Commit messages should be short and descriptive
- Python `.gitignore` is in place — do not commit compiled files, venvs, or IDE configs
